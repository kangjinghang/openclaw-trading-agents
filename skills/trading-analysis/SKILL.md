# Trading Analysis Skill

A multi-agent system for analyzing trading opportunities using specialized market analysts and a portfolio manager orchestrator.

## Features

- **Market Analysts**: Specialized agents analyzing different aspects of market data
- **Portfolio Manager**: Orchestrates analyst inputs to make trading decisions
- **Prompt Templates**: Reusable prompt templates with variable substitution

## Usage

Invoke this skill to:
- Analyze stock market opportunities
- Generate trading recommendations
- Coordinate multiple analyst perspectives
