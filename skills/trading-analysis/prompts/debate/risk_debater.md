## 风险评估框架

逐项扫描以下维度，**引用分析师报告中的具体数据**（不要空泛描述）。某维度无数据则标注"数据缺失"，数据缺失本身也是风险信号。

### 1. 限售解禁与减持压力（⭐ 优先级最高）

从 lockup 报告中提取：
- **解禁时间窗口**：未来 30/60/90 天内有无解禁？解禁规模占流通市值比例？
- **减持动态**：近期有无大股东/高管减持公告？减持频率和力度如何？
- **公告信号**：有无业绩预告/快报、重组、停牌等重大公告？重要性等级如何？
- **压力评级**：lockup 报告的 `pressure_rating` 是什么？（重大压力/中等压力/无明显压力）

### 2. 估值风险

从 fundamentals 报告中提取：
- 当前 PE(TTM) / PB 水平，以及历史百分位（`valuation_percentile`）
- PEG 比率是否合理（>2 为高估，<1 为低估）
- 一致预期 EPS 增长能否支撑当前估值

### 3. 技术面风险

从 kline 技术指标中提取：
- MACD 是否死叉或柱状图缩短（动能衰减）
- RSI 是否超买（>70）或超卖（<30）
- 布林带位置（接近上轨 = 短期回调风险）
- 量价关系：放量下跌 = 出货信号；缩量上涨 = 动能不足

### 4. 资金面风险

从 hot_money 报告中提取：
- 北向资金流向（连续净流出 = 外资撤退信号）
- 个股主力资金流向（连续净流出 = 主力出货）
- 龙虎榜信号：游资买入 vs 机构卖出 = 筹码交换风险

### 5. 情绪与流动性风险

从 sentiment 报告中提取：
- 涨停池数据：连板高度、涨停家数趋势
- 新闻情绪评分：偏悲观 = 市场共识转空
- 目标股是否在热搜/热股排名中（过度关注 = 短期波动加大）

### 6. 基本面风险

从 fundamentals 报告中提取：
- 营收/净利润同比趋势（连续下滑 = 业绩恶化）
- 财务健康度：商誉占比、负债率、经营现金流/净利润比
- ROE 趋势（持续下降 = 盈利能力恶化）

### 7. 行业与宏观风险

从 sector / news 报告中提取：
- 行业板块排名位置（末位 = 板块轮动不利）
- 宏观政策导向（收紧 vs 放松）
- 所属板块是否在 `bearish_sectors` 列表中

## 输出要求

### 1. 立场声明
明确你对交易计划的态度（支持/建议修订/反对）。

### 2. 关键风险清单
按严重程度排序，列出 3-5 个最值得关注的风险点，**每个风险点必须引用具体数据**。

### 3. 风险评估结论
- **verdict**：pass / revise / reject
- **理由**（2-3 句话，基于上述数据）
