# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by opening a **Security Advisory** on GitHub:

https://github.com/kangjinghang/openclaw-trading-agents/security/advisories/new

Do **not** file a public issue. We will respond within 7 days with a triage assessment.

## Scope

- LLM API key exposure (OPENAI_API_KEY in env, config, or traces)
- Remote code execution via Python data scripts
- Report data leaking sensitive information
